## コマンド一覧

| コマンド                                     | 機能                        |
|------------------------------------------|---------------------------|
| /absence                                 | 自身の不在状態を確認します。            |
| /absence check <player>                  | 指定したプレイヤーの不在状態を確認します。     |
| /absence list                            | 不在中のプレイヤー一覧を表示します。        |
| /absence-admin add <player> <YYYY-MM-DD> | 指定したプレイヤーの不在期間を管理者が追加します。 |
| /absence-admin del <player>              | 指定したプレイヤーの不在期間を管理者が削除します。 |
